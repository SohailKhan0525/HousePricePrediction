# AI Model for House Price Prediction

This project features an AI model designed to predict house prices.

## How to Use

Click the following link to access the interactive application: [House Price Prediction App](https://housepriceprediction-rqo8ykwgldxbpemwl6kvok.streamlit.app/)

## What I Learned

Developing this house price prediction model provided valuable experience in several key areas:

*   **Data Cleaning:** Techniques to handle missing values, inconsistencies, and errors within the dataset.
*   **Exploratory Data Analysis (EDA):** Methods for understanding data patterns, identifying relationships, and making informed decisions.
*   **Model Tuning:** Strategies for optimizing model parameters to achieve better performance and accuracy.
*   **Outlier Removal:** Approaches to identify and address outliers that could skew the model's predictions.
